Tags

Strong / Negrito

O Strong ou >B< é uma tag para dar negrito (strong) nas palavras em questão. Muito utilizado para destacar textos ou títulos.

Emphasis / itálico.

O Emphasis ou >I< é uma tag para dar itálico às palavras, também dando destaques às mesmas quando necessário.

Mark / marca texto

O Mark é uma tag que serve como marcador de texto para as palavras, marcando algo de suma importância para o leitor em questão, como por exemplo; isso aqui.

Há, também, a possibilidade de marcar todas as tags com uma cor específica ao usar o css dando, sem a necessidade de abrir um bg color na própria tag, e sim no documento inteiro; esse texto estará rosa, por exemplo

Textos grandes e pequenos

Também há textos grandes e pequenos, como por exemplo: Pequeno & Grande.

Texto deletado

Posso marcar um texto como excluído para indicar ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois
